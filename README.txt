Сервер NatICQ
(c) Rst7, BoBa, Kibab
=====================
Использование:
	./ig11 5050 путь_к_файлу_статистики
В директории с сервером лежит файл stat_template.txt, в который необходимо положить шаблон страницы с информацией о количестве подключённых клиентов.
Его содержимое будет записываться в файл_статистики. Файл статистики -- текстовый файл, в котором "%d" будет заменяться на число клиентов.

Пример запуска на "боевом" сервере:
#!/bin/sh
cd igserver
/usr/sbin/daemon -u nobody ./ig11 5050 /path/to/www/clients_count.html > ig_log.txt

Багрепорты по работе сервера принимаются только при указании точной версии ОС и сервера, а также используемой версии NatICQ.
